# devsolaris-natursur
Portal web de la empresa devSolaris de NaturSur

Para cada uno de estos pasos, utilizar un terminal distinto. 
Las 3 instancias de backend, bot y frontend se deben de ejecutar simultáneamente

# Setup backend
IMPORTANTE estar en la carpeta devsolaris-backend/
-Ejecutar mvn install
-Ejecutar mvn spring-boot:run

# Setup bot:
IMPORTANTE estar en la carpeta devsolaris-bot/
-Ejecutar pip install -r requirements.txt
-Abrir el archivo bot.py, y ejecutarlo (hacer click en botón de play arriba a la derecha)

# Setup frontend
IMPORTANTE estar en la carpeta devsolaris-frontend/
-Ejecutar npm install en terminal
-Ejecutar npm run en terminal
